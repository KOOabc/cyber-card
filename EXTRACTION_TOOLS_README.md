## ❓ 常见问题

**Q: 为什么提取速度较慢？**  
A: 因为需要逐个打开邮件提取完整内容，每封约2-3秒。这是必要的，才能获取完整的用户信息。

**Q: 可以提取超过50封邮件吗？**  
A: 可以！修改脚本中的 `Math.min(threads.length, 50)` 改为更大的数字，或删除限制。

**Q: 提取的数据不完整怎么办？**  
A: 确保邮件格式正确，或使用"邮件解析工具"手动处理。

**Q: 能否自动发送到飞书/钉钉？**  
A: 目前仅支持本地下载，可以导出后手动上传到协作工具。
