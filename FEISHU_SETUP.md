# 数据收集配置指南

## 🎯 方案1：复制到剪贴板（推荐，零配置）

**完全免费，无需任何配置！**

### 工作原理
用户点击「确认，生成名片」后，所有表单数据会自动复制到剪贴板，格式化为易读的文本。

### 使用方法
1. 用户提交表单
2. 系统自动复制数据到剪贴板
3. 您打开飞书多维表格/腾讯文档/Excel
4. 直接 `Ctrl+V` (Windows) 或 `Cmd+V` (Mac) 粘贴

## 🚀 方案2：飞书群聊机器人（需配置）

如果您需要自动推送到飞书群聊，可以配置 Webhook：

### 第一步：创建飞书群聊机器人

1. 打开**飞书 App**，创建一个新的群聊（或使用现有群聊）
2. 点击群聊右上角「...」→ 「设置」
3. 选择「群机器人」→ 「添加机器人」
4. 选择「自定义机器人」
5. 设置机器人名称（如「名片数据收集」）和描述
6. 点击「添加」后，**复制 Webhook 地址**

Webhook 地址格式类似：
```
https://open.feishu.cn/open-apis/bot/v2/hook/xxxxxxxxxxxxxxxxxx
```

### 第二步：在代码中配置 Webhook

打开 `src/App.tsx` 文件，找到第 **445** 行：

```typescript
const FEISHU_WEBHOOK = 'YOUR_FEISHU_WEBHOOK_URL_HERE'
```

替换为您刚才复制的 Webhook 地址：

```typescript
const FEISHU_WEBHOOK = 'https://open.feishu.cn/open-apis/bot/v2/hook/xxxxxxxxxxxxxxxxxx'
```

### 第三步：重启开发服务器

```bash
npm run dev
```

配置后，数据会自动推送到飞书群聊，同时也会复制到剪贴板作为备份。
